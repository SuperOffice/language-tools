# SuperOffice Extension for Visual Studio Code

This extension includes a viewContainer for working with customizations in SuperOffice. It enables developers to easily view, download and upload scripts from/to a tenant.

## Features

- View scripts
- View extratables and fields
- Upload and download scripts
